# Codeplg

## Run it Now!
- Clone this repo and Pip install needed packages
- Run these in the command line
```
set FLASK_APP=flaskr
set FLASK_ENV=DEV
flask init-db
flask run
```
- Open 127.0.0.1:5000 in the Browser
- Modify Anything and Pull Request!
